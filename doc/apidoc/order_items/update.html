PUT /orders/:order_id/items/:id
Updates order item with :id

Errors

404 Not found.
422 Missing parameter

Examples

PUT /orders/13/items/12
port=123&host=www.example.com
204

Params

Param name Description
id
required
Value: Must be a number.
order_id
required
Value: Must be a number.
port
optional
Value: Must be a number.
host
optional
Value: Must be String
provision_status
optional
Value: Must be one of: ok, warning, critical, unknown, pending.
