POST /alerts/sensu
Create new Sensu alert.

Endpoint to push an alert from Sensu to Core.
If an alert already exists for this order, then update that order.

Errors

422 Missing parameter

Examples

POST /alerts/sensu
hostname=foo.bar.com&port=5000&service=postgresql&status=OK&message=This+is+a+test&event=tbd

required
