GET /staff/:id/settings/:user_setting_id
Shows user settings detail

Errors

404 Not found.

Examples

GET /staff/39/settings/4
200
{
  "id": 4,
  "staff_id": 39,
  "name": "test user setting",
  "value": "test user value",
  "created_at": "2015-01-08T13:35:57.862Z",
  "updated_at": "2015-01-08T13:35:57.864Z",
  "deleted_at": null
}

staff_id
