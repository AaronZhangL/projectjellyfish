POST /user_setting_options
Create new setting that all users will have.

Examples

POST /user_setting_options?field_type=bar&help_text=fizz&label=foo&options=buzz&required=true
201
{
  "id": 2,
  "label": "foo",
  "field_type": "bar",
  "help_text": "fizz",
  "options": "buzz",
  "required": true,
  "created_at": "2015-01-08T13:35:59.634Z",
  "updated_at": "2015-01-08T13:35:59.634Z",
  "deleted_at": null
}

Params

Param name Description
label
required

The label of the setting. Varchar(255)


Value: Must be String
field_type
required

The field type (text, textarea, radio, checkbox). Varchar(100)


Value: Must be String
help_text
required

Inline help text to display. Varchar(255)


Value: Must be String
options
required

JSON for options for field_type that needs them (like radio, checkbox, etc). Text BLOB.


Value: Must be String
required
required

Whether option is mandatory. Boolean


Value: Must be String
