PUT /projects/:id
Updates project with :id

Errors

404 Not found.
422 Missing parameter

Examples

PUT /projects/9
name=Created&description=description&cc=cc&staff_id=staff_id&budget=1&start_date=2015-01-08&end_date=2015-01-15&approved=Y&img=img&project_answers[][project_question_id]=8&project_answers[][answer]=To+get+to+the+other+side.&includes[]=project_answers
200
