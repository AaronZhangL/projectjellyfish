Alerts

Resource Description
GET /alerts Returns all active alerts. (Default behavior)
GET /alerts/:id Shows alert with :id
GET /alerts/all Shows all alerts, both active and inactive.
GET /alerts/active Shows all active alerts. Where end_date is null or has yet to occur and start is null or has already occurred.
GET /alerts/inactive Shows all inactive alerts. Where end_date has already occurred or start_date has yet to occur.
POST /alerts/sensu Create new Sensu alert.
POST /alerts Creates a new alert
PUT /alerts/:id Updates alert with given :id
DELETE /alerts/:id Deletes alert with :id

Staff

Resource Description

DELETE /staff/:id/settings/:user_setting_id Deletes a user setting from a staff member.
